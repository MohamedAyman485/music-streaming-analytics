# 🎵 Music Streaming Analytics Dashboard

## 📌 Project Overview
This project analyzes music streaming data using Power BI to uncover user behavior, listening trends, and track performance.

## 🛠 Tools Used
- Power BI
- Power Query
- DAX

## 📊 Dashboard Pages
- Home
- Overview
- User Analytics
- Track Analytics
- Geo Analytics

## 📈 Key Insights
- User listening trends
- Top tracks and artists
- Geographic distribution
- Device usage analysis
- Listening behavior over time

## 📂 Dataset
The project uses a star schema including:
- Fact Listens
- Dim User
- Dim Track
- Dim Date
- Dim Device
- Dim Geo
